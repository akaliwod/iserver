# Get Action properties

Action property in resource is used to define an operation that a client can perform by submitting a POST request to the URI specified by the Action.

iserver makes it every easy to find actions for URI defined resource as well as searching recursively for all action properties.

## Get system action

```
# iserver get redfish endpoint
    --type ucsc
    --ip 10.58.50.42
    --username admin
    --password ******
    --uri Systems/SYSTEM_ID
    --action
```
